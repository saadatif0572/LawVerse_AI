# LawVerse_AI

A modern React + Vite project for AI-powered legal assistance. Includes:
- Dark/light theme with a global toggle
- Animated hero and pages (text + background)
- Firestore-backed lawyer details and services
- Tailwind CSS styling

## Development

Install and run:

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
